Help & Quick Guide

    

      

        

          
Getting started

          
    
            
  • Load files: Import (JSON/HAR), or drag & drop anywhere.
    • 
            
  • Multi‑file: each file opens as a workspace (top bar).
    • 
            
  • Theme: toggle light/dark at top‑right.
    • 
          

          
Navigation

          
    
            
  • Tree: expand/collapse, double‑click keys to jump.
    • 
            
  • Breadcrumb: click any token (e.g. $, .items) to navigate.
    • 
            
  • Back / Forward: use the ◀ ▶ buttons near the breadcrumb.
    • 
          

          
Search & JSONPath

          
    
            
  • Global Search: enter a term and press Search (or Ctrl/Cmd+F).
    • 
            
  • Advanced: build multiple conditions, AND/OR logic.
    • 
            
  • JSONPath: Query → presets or your own expression (e.g. $.log.entries[*].request.url), copy results.
    • 
          

          
HAR Intelligence

          
    
            
  • Waterfall: durations + phases per request; filter by method/errors/slow or URL.
    • 
            
  • Auth Flows, Cookies, Security, Domains, Duplicates, GraphQL.
    • 
            
  • Open any request to view full detail.
    • 
          

          
Actions

          
    
            
  • Copy/Raw: copy selected JSON or open Raw pane.
    • 
            
  • Schema: infer JSON schema for current path.
    • 
            
  • Diff: compare two JSON blobs.
    • 
            
  • Export: JSON/CSV/Excel/Markdown/HTML for selected results.
    • 
 
  • IR Rendering: toggle via toolbar 'IR Render' (beta). Keep OFF for v3.1 parity; turn ON to test IR-backed rendering.
    • 
 
  • Advanced features: use the ⚙️ Advanced control near the Search bar to enable/disable deepSearch, schema, fullTree.
    • 
 
  • IR Rendering: toggle via toolbar 'IR Render' (beta). Keep OFF for v3.1 parity; turn ON to test IR-backed rendering.
    • 
          

        

        

          
Keyboard

          
    
            
  • Ctrl/Cmd + F: focus Search.
    • 
            
  • Esc: close open modals or Advanced panel.
    • 
          

          
Tips

          
    
            
  • Use Sort Keys to alphabetize objects in the tree.
    • 
            
  • Double‑click a key in the tree to open that path directly.
    • 
            
  • In tables, click headers to sort; double‑click a row to open its path.
    • 
          

          

            
Privacy

            
Fully offline. No data leaves your device.
